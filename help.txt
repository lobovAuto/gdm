Программа для выкачивания зависимостей из репозиториев
В начале каждого файла обязательно должно быть указание
на систему удаленного управления репозиториями
Пример:
ROOT_RMS ssh://git@git.redpioneer.ru:2224
Так же можно добавить корневую папку для выкачиваемых зависимостей
Это значит, что при выкачивании компонента
путь, указанный в корневой зависимости, будет опускаться
Пример:
ROOT_FOLDER //embedded
Символ комментария строки - #
Синтаксис подтяжки зависимости:
LINK logic/drivers/stepper.git main/head force
Где LINK - указатель на то, что далее следует ссылка на компонент
logic ... git - ссылка на компонент. В данном случае, начальный путь embedded опущен
main/head - ветка и комит
Если указать только ветку, то выкачается последний комит
Вместо комита можно указывать тег
Хинт force нужно добавлять, если несколько компоннетов
ссылаются на разные версии другого компонента.
Может быть вызван только в главном файле
